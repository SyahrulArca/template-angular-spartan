# Base Template Angular + Spartan — Task List

> Tanggal: 2026-07-14  
> Scope: Menyempurnakan project `template-angular-spartan` dari setup infra menjadi base template siap fork.

**Status:** Required items 1–10 selesai (2026-07-14).

---

## Required

Urutan pengerjaan mengikuti rekomendasi — selesaikan secara berurutan.

### 1. Commit baseline setup Spartan

Semua setup Spartan saat ini masih untracked/modified (`libs/`, `components.json`, `.agents/`, perubahan `package.json`, `tsconfig`, `styles.css`, dll.). Commit sebagai baseline agar template bisa di-clone dan langsung jalan.

### 2. Selaraskan komponen (disk vs CLI)

Komponen ada di disk tapi tidak terdeteksi CLI: `sidebar`, `typography`, `alert`, `aspect-ratio`. Selaraskan via CLI (`ng g @spartan-ng/cli:ui --name=<component>`) atau hapus yang orphan — hindari duplikat/konflik saat re-run generator.

### 3. Wire `provideSpartanHlm()` di `app.config.ts`

Tambahkan provider dari `@spartan-ng/helm/utils` supaya overlay CDK (sheet, tooltip, dialog) tidak konflik dengan `position: fixed` di Angular 21.

### 4. Bangun layout shell (sidebar + header + content area)

Compose layout dasar menggunakan komponen Spartan yang sudah ada: `hlm-sidebar-*`, `hlm-sidebar-inset`, trigger, menu, separator. Pertimbangkan Spartan block `sidebar` via MCP/CLI untuk mempercepat.

### 5. Setup routing structure

Lazy feature routes, redirect default, wildcard 404. `app.routes.ts` saat ini masih kosong.

### 6. Ganti placeholder Angular default

Hapus landing page default Angular CLI di `app.html` (300+ baris inline CSS + logo Angular). App layer harus memakai Spartan, bukan placeholder.

### 7. Fix asset gaps

- `app.ts` mereferensikan `app.css` yang tidak ada — buat file atau hapus referensi.
- `index.html` mereferensikan `favicon.ico` — folder `public/` kosong.

### 8. Dark mode toggle

CSS variables dark sudah ada (`:root.dark` di `styles.css`), tapi belum ada service/util toggle, class `dark` di `<html>`, dan UI toggle (button/switch).

### 9. Install komponen UI pendukung layout

Belum ter-install, disarankan untuk template UX:

- `breadcrumb`, `avatar`, `dropdown-menu` (user menu)
- `sonner` (toast), `spinner` (loading)
- `card` (content wrapper)

### 10. Update README template

Ganti README default Angular CLI. Dokumentasikan: cara setup, struktur folder, cara menambah komponen Spartan, dan cara menjalankan project.

## Optional

Item di bawah ini meningkatkan kualitas template ke production-grade — kerjakan setelah required selesai.

### 11. `provideHttpClient()` + interceptor placeholder

Setup HTTP client di `app.config.ts` dengan interceptor skeleton untuk auth/error handling.

### 12. Environment files

Tambahkan `environment.ts` / `environment.development.ts` untuk konfigurasi per-environment.

### 13. Auth guard placeholder + login route skeleton

Route guard dasar dan halaman login skeleton sebagai starting point integrasi auth.

### 14. ESLint / Prettier config

Perluas konfigurasi linting/formatting selain `.prettierrc` yang sudah ada.

### 15. CI workflow (GitHub Actions)

Pipeline build + test otomatis saat push/PR.

### 16. Align `@angular/cdk` ke 21.2.x

Saat ini `@angular/*` di `^21.2.0` tapi `@angular/cdk` pinned di `21.0.0` — samakan versi.

### 17. `.cursor` rule untuk Spartan conventions

Tambah rule Cursor khusus Spartan (`hlm()`, semantic colors, `provideIcons`, composition rules) — `.cursor/rules/cursor.mdc` saat ini hanya Angular generic.
